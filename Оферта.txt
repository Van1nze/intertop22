=== Политика конфиденциальности ТОО «ИНТЕРТОП ЦЕНТРАЛЬНАЯ АЗИЯ» ===

== Общие положения ==
- Политика конфиденциальности является частью Публичной оферты.
- Применяется ко всем данным пользователей сайта intertop.kz и мобильного приложения «Интертоп».
- Использование сайта/приложения означает согласие с Политикой.
- В случае несогласия — необходимо прекратить использование.

== Термины ==
- Администрация сайта — сотрудники или третьи лица, уполномоченные обрабатывать данные.
- Данные — любая информация, включая персональные данные.
- Обработка данных — сбор, хранение, использование, передача, удаление и др.
- Конфиденциальность — запрет на разглашение данных без согласия субъекта.
- Cookies — фрагменты данных, сохраняемые браузером.
- IP-адрес — уникальный сетевой адрес устройства.

== Какие данные собираются ==
- При регистрации и заказах: ФИО, дата рождения, телефон, e-mail, адрес доставки и др.
- Автоматически: IP, cookies, браузер, время доступа, посещенные страницы, реферер.
- Через приложение: данные устройства, геолокация (если разрешена), технические ошибки, метрики производительности.

== Цели обработки данных ==
- Регистрация и идентификация пользователя.
- Доступ к персонализированным ресурсам и бонусному клубу.
- Поддержка связи: уведомления, SMS, e-mail.
- Обработка заказов.
- Клиентская и техническая поддержка.
- Рекламные рассылки, акции, спецпредложения.
- Аналитика и статистика.

== Обработка и хранение ==
- Данные обрабатываются в соответствии с законодательством РК.
- Сроки хранения определяются согласием пользователя.

== Обязанности сторон ==
Пользователь:
- предоставляет корректные и достоверные данные;
- обновляет их при изменениях;
- самостоятельно защищает доступ к своим данным.

Администрация:
- использует данные только по целям политики;
- не разглашает данные (кроме случаев, предусмотренных законом);
- блокирует недостоверные данные по запросу.

== Ответственность ==
- Администрация отвечает за умышленное разглашение данных.
- Не несёт ответственности, если данные:
  • стали публичными ранее;  
  • получены от третьих лиц;  
  • получены через несанкционированный доступ;  
  • разглашены с согласия пользователя.

== Споры ==
- Обязателен досудебный порядок урегулирования (15 дней на ответ по претензии).
- В случае недостижения соглашения — спор решается в суде РК.

== Заключительные положения ==
- Компания вправе вносить изменения в Политику без согласия пользователя.
- Новая версия вступает в силу с момента публикации на сайте или в приложении.
